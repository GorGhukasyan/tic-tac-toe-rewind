# Tic Tac Toe Game

This is a simple Tic Tac Toe game built using Lovable.ai. You can play the classic X and O game against another player. The game features an intuitive interface and keeps track of moves for a fun and interactive experience.

## How to Play
1. Clone the repository: `git clone https://github.com/yourusername/your-repo.git`
2. Open `index.html` in your browser.
3. Click on the squares to place X or O.
4. Try to get three in a row to win!

## Technologies Used
- HTML
- CSS
- JavaScript
- Lovable.ai (for game logic)
